# Fonte Small Cake

Para usar a fonte Small Cake no projeto:

1. Baixe a fonte Small Cake do site: https://www.dafont.com/small-cake-2.font
2. Extraia o arquivo ZIP
3. Converta o arquivo .ttf ou .otf para .woff2 e .woff usando uma ferramenta online como:
   - https://cloudconvert.com/ttf-to-woff2
   - https://convertio.co/ttf-woff2/
4. Coloque os arquivos convertidos nesta pasta:
   - SmallCake.woff2
   - SmallCake.woff
   - SmallCake.ttf (opcional, como fallback)
   - SmallCake.otf (opcional, como fallback)

A fonte será carregada automaticamente após adicionar os arquivos.
